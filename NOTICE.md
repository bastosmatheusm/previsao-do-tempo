# NOTICE

## Previsão do Tempo

Este projeto utiliza os seguintes serviços e bibliotecas de terceiros:

---

### Open-Meteo API
- **Descrição:** API gratuita de previsão do tempo e geocodificação.
- **Site oficial:** https://open-meteo.com/
- **Licença:** [Attribution 4.0 International (CC BY 4.0)](https://creativecommons.org/licenses/by/4.0/)
- **Uso neste projeto:** Obtenção de dados meteorológicos atuais, previsão de 7 dias e conversão de nomes de cidades em coordenadas geográficas.
- **Atribuição:** Os dados meteorológicos são fornecidos pela Open-Meteo (https://open-meteo.com/).

---

### Jest
- **Descrição:** Biblioteca de testes unitários para JavaScript.
- **Site oficial:** https://jestjs.io/
- **Licença:** [MIT License](https://github.com/jestjs/jest/blob/main/LICENSE)
- **Uso neste projeto:** Execução de testes automatizados das funções de integração com a API.
